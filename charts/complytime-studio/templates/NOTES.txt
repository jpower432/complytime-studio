ComplyTime Studio {{ .Chart.AppVersion }} installed in namespace {{ .Release.Namespace }}.

Prerequisite: kagent >= 0.8.0 with KMCP controller must be installed in this cluster.

Architecture:
  Assistant:     BYO ADK agent (Python) — studio-assistant
  Gateway:       Go service — serves workbench SPA, REST APIs, A2A proxy
  MCP servers:   gemara-mcp, oras-mcp — deployed as MCPServer CRDs via KMCP
{{- if .Values.clickhouse.enabled }}
  ClickHouse:    Evidence store + clickhouse-mcp MCP server
{{- end }}

Access:
  Gateway:       kubectl port-forward -n {{ .Release.Namespace }} svc/studio-gateway 8080:8080
{{- if .Values.registry.enabled }}
  OCI registry:  kubectl port-forward -n {{ .Release.Namespace }} svc/studio-registry {{ .Values.registry.service.port }}:{{ .Values.registry.service.port }}
{{- end }}

Local development:
  go run ./cmd/gateway     # runs gateway (needs GEMARA_MCP_URL, ORAS_MCP_URL)

Tear down:
  helm uninstall {{ .Release.Name }} -n {{ .Release.Namespace }}
